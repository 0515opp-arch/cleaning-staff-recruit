## お問い合わせフォーム（二重化で障害に強い構成）

送信は次の順で自動的に処理されます。

1. **優先**：[FormSubmit](https://formsubmit.co/)（無料・サーバー不要）へ非同期送信 → 成功すれば画面内で完了表示。
   - 送信先メール：`masaki0515opp@gmail.com`
   - **初回のみ**：最初の送信時に FormSubmit から確認メールが届くので、本文のリンクをクリックして有効化してください（以降は自動でメール受信）。
2. **フォールバック**：FormSubmit が応答しない／障害（例: Cloudflare 521）の場合は、**訪問者のメールアプリを入力内容入りで自動起動**（`mailto:`）。そのまま送信すれば届きます。
3. **常時導線**：フォーム下部に「お電話」「メール」リンクを常設。電話 070-2294-0290 はいつでも有効。

> これにより、他社フォームサービスが落ちていても応募経路が途切れません。
